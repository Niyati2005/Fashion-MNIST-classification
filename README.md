# Fashion MNIST Classification

## 👩‍💻 Author
Niyati Reshmi

## 📌 Project Overview
This project uses Deep Learning and Artificial Neural Networks (ANN) to classify Fashion MNIST images into different clothing categories.

The model is built using TensorFlow and Keras and trained on the Fashion MNIST dataset.

---

## 🚀 Technologies Used

- Python
- NumPy
- Matplotlib
- TensorFlow
- Keras
- Scikit-Learn
- Seaborn

---

## 📊 Dataset

Fashion MNIST Dataset contains 70,000 grayscale images of clothing items.

Classes include:

- T-shirt/Top
- Trouser
- Pullover
- Dress
- Coat
- Sandal
- Shirt
- Sneaker
- Bag
- Ankle Boot

---

- Neural Networks
- Dense Layers

- Softmax Activation
- Dropout
- Adam Optimizer
- Loss Function
- Accuracy Evaluation
- Confusion Matrix

## 📈 Model Performance

The model was trained using TensorFlow and achieved good classification accuracy on test data.
